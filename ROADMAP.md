# 迭代规划

## 当前状态

桌面图片查看器（Tauri v2 + React 19 + Rust），已完成 12 个迭代提交。

### 已实现

- 图片查看（canvas 渲染、高 DPI、缩放/拖拽、双击全屏）
- 文件夹浏览（侧边栏文件树、子文件夹切换、返回上级）
- 缩略图列表（sips 生成、懒加载、IntersectionObserver）
- 可拖拽调整侧边栏宽度（140px - 500px，持久化）
- 暗色/浅色主题切换（Obsidian / Paper）
- 中英文切换
- 键盘快捷键（方向键切换图片、F 全屏、0 重置缩放、Cmd+/- 缩放、Cmd+B 切换侧边栏）
- 图片预加载（前/后一张图缓存，切换无白屏）
- macOS「打开方式」集成（冷启动 + 热启动）

### 技术栈

| 层 | 技术 |
|----|------|
| 桌面框架 | Tauri v2 |
| 前端 | React 19 + TypeScript 5.8 + Vite 7 |
| 后端 | Rust (serde, urlencoding) |
| 缩略图 | macOS sips CLI |
| 包管理 | pnpm |
