# Spicy Chicken Preview

[中文版](./README.zh.md) | [English](./README.md)

基于 Tauri 2 + React 19 + TypeScript 构建的 macOS 原生图片查看器。简约、快速、键盘友好。

## 功能

- **图片浏览** — 打开单张图片或浏览整个文件夹，基于 Canvas 渲染，支持 Retina 显示
- **侧边栏缩略图列表** — 通过懒加载缩略图浏览文件夹内所有图片
- **缩放与平移** — 支持双指缩放、滚轮缩放、拖拽平移，双击切换全屏
- **全屏模式** — 沉浸式全屏体验，底部缩略图条自动隐藏
- **幻灯片** — 自动播放，可切换间隔（2s / 3s / 5s / 10s），带交叉淡入淡出过渡动画
- **图片预加载** — 后台预加载相邻图片，实现即时切换
- **排序** — 支持按名称、尺寸、长宽比、修改时间排序，升降序切换
- **状态栏** — 显示图片尺寸、文件大小、格式和当前缩放比例
- **拖拽打开** — 从访达直接拖拽图片到窗口打开
- **主题切换** — 支持深色 / 浅色主题
- **国际化** — 支持中文 / 英文语言切换
- **设置面板** — 集中管理主题、语言、幻灯片间隔、侧边栏宽度等偏好设置
- **最近文件夹** — 快速访问最近打开过的文件夹，重启后依然保留
- **自适应工具栏** — 窄窗口下工具栏项目自动折叠到溢出菜单，悬停展开全部
- **文件关联** — 支持 jpg、jpeg、png、gif、webp、bmp、svg、avif、tiff、tif 格式

## 键盘快捷键

| 按键 | 操作 |
| --- | --- |
| ← / → | 上一张 / 下一张 |
| 空格 | 切换幻灯片 |
| F | 切换全屏 |
| Ctrl/Cmd + =/+ | 放大 |
| Ctrl/Cmd + - | 缩小 |
| Ctrl/Cmd + 0 | 重置缩放 |
| Ctrl/Cmd + B | 切换侧边栏 |

## 开发

```bash
pnpm install
pnpm run dev
```

Built with [Tauri](https://tauri.app/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/).
